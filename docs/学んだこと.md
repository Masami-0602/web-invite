## Prettierのインストール

- `npm install --save-dev prettier`でプロジェクトにインストール
- setting.jsonに以下の設定を追加する

```
{
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "[typescript]": {
    "editor.defaultFormatter": "esbenp.prettier-vscode"
  },
  "[typescriptreact]": {
    "editor.defaultFormatter": "esbenp.prettier-vscode"
  },
  "[javascript]": {
    "editor.defaultFormatter": "esbenp.prettier-vscode"
  },
  "[json]": {
    "editor.defaultFormatter": "esbenp.prettier-vscode"
  }
}
```

- ※.setting.jsonは最初からはないので、.vscode/.setting.jsonで作成する
- 実行コマンド

```
npx prettier --write .
```

- ※package.jsonに以下を追加すると、`npm run format`で実行できる

```
{
  "scripts": {
    "format": "prettier --write ."
  }
}
```

## fontの適用方法

- プロジェクト内の適切な場所（例：src/assets/fonts）に.ttfファイルを入れる
- グローバルCSSファイル（src/App.css or src/index.css等）に以下を追加

```CSS
@font-face {
  font-family: 'Hafidz';
  src: url('.assets/fonts/Hafidz.ttf') format('truetype');
  font-weight: normal;
  font-style: normal;
}
```

## Typographyのカスタムvariantの設定方法

- src/theme.d.tsを作成し、以下を記述

```typescript
import "@mui/material/styles";

declare module "@mui/material/styles" {
  interface TypographyVariants {
    weddingTitle: React.CSSProperties;
  }

  interface TypographyVariantsOptions {
    weddingTitle?: React.CSSProperties;
  }

  declare module "@mui/material/Typography" {
    interface TypographyPropsVariantOverrides {
      weddingTitle: true;
    }
  }
}
```

- MUIのcreateTheme()でカスタムvariantを追加

```typescript
import { createTheme } from "@mui/material/styles";

export const theme = createTheme({
  typography: {
    fontFamily: "'Noto Serif JP', serif",
    weddingTitle: {
      fontSize: "26px",
      fontWeight: 500,
      fontFamily: "'Hafidz', cursive",
    },
    // 他にも追加可能
    messageBody: {
      fontSize: "14px",
      fontFamily: "'Noto Serif JP', serif",
    },
  },
});
```

- ThemeProviderでアプリ全体に適用

```typescript
import React from 'React';
import ReactDOM from 'react-dom';
import { ThemeProvider } from '@mui/material/styles';
import { theme } from './theme';
import App from './App';

ReactDom.render(
  <ThemeProvider theme={theme}>
    <App />
  </ThemeProvider>
  document.getElemenetById('root')
);
```

## フロントエンド

### 表示方法

- 要素を横に均等に並べるには
  - 親要素に`display:flex`を適用し、子要素に`flex:1`を指定する

- `width: "100%"`にすると、下の要素がスクロールバーと被ってしまう
  - するロールバーの幅（約15px前後）を含めて計算されている為
    - width: calc(100% - 8px)
    - maxWidth: calc(100% - 16px)
    - scrollbar-gutter: stable
